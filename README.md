# 🕒 Timestamp Converter

심심풀이로 시작한 CLI 기반 타임스탬프 변환기입니다.  
Unix Timestamp(초 단위)를 사람이 읽기 쉬운 날짜/시간 형식으로 변환합니다.

---

## ✨ 기능

- 현재 시간의 Unix Timestamp 출력
- Unix Timestamp → 날짜/시간 변환
- UTC / KST(UTC+09:00) 동시 출력

---

## 🖥 실행 방법

```bash
python timestamp_converter.py
```
